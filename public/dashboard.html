📖 System Dossier & Overview

Ghost in the Machine (Project Phantasm) is an advanced, high-fidelity Linux terminal deception honeypot designed to trap, profile, and study malicious actors in real-time. By emulating a realistic SSH environment, it tricks attackers into executing command lines while keeping them completely sandboxed from any real target system.

🛡️ Core Deception Systems

  • Realistic Shell Emulation: Emulates Linux shell behaviors, command history, simulated file systems, and authentic server components (such as sudo prompts, configuration dumps, and system files).
  • Adversary Profiling Engine: Dynamically calculates risk scores based on command complexity, payload syntax, and attempts to access privileged files.
  • Keystroke & Password Tarpit: Captures attacker authentication attempts (usernames/passwords) and files uploaded to decoy services.
  • AI Poisoning Strategy: Feeds decoy, highly structured pseudo-data to AI scanners, disrupting hostile automated intelligence gathering.
